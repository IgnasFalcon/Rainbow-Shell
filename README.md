A shell script to make a rainbow!
# How to run
## On Windows
### PowerShell / Terminal (Recommended)
<pre>
./rainbow.bat
</pre>
### File Explorer

1) Double-click the "rainbow" file in the root directory

OR

2) Right-click the file and open

OR

3) Select the file and press ENTER
## On Linux
## Bash / Terminal
<pre>
bash rainbow.sh
</pre>
