# Rainbow-Shell
A script to make a rainbow!
# 1. Clone repository
## Windows & Linux
### Terminal:
<pre>git clone https://github.com/IgnasFalcon/Rainbow-Shell.git</pre>
<pre>cd Rainbow-Shell</pre>
Got an error here? This link can help you: https://github.com/IgnasFalcon/Rainbow-Shell/issues/1
# 2. Run the program
## Windows
### PowerShell / Terminal (Recommended)
<pre>
./rainbow.bat
</pre>
### File Explorer (Not recommended)
Double-click the "rainbow" file while in the root directory
## Linux
### Bash / Terminal
<pre>
bash rainbow.sh
</pre>
You can even set how many cycles of the rainbow it runs! (This feature only works for Linux for now)
<pre>bash rainbow.sh 3</pre>
This goes through all seven colors three times!
# 3. Install (Linux only)
This step is technically optional, but it is **highly advised** to do so.
### Get into the root directory
If you didn't originally run this command, you should do it now.
<pre>cd Rainbow-Shell</pre>
### Run the installer
Run this to install the package. **Make sure you are in the root directory! Run the command above to get there.**
<pre>bash install.sh</pre>
### If everything worked out, you can now run it with this syntax:
<pre>rainbow [-l] [CYCLES]</pre>
Cycles is the number of times the code will go through the full rainbow. To make it display in a list, use [-l]
