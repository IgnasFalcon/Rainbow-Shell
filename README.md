# Rainbow-Shell![License](https://img.shields.io/badge/license-MIT-green)![Shell](https://img.shields.io/badge/language-bash-blue)
A shell script to make a rainbow!
## How to run
### On Windows
PowerShell / Terminal
<pre>
./rainbow.bat
</pre>
### On Linux
Bash / Terminal
<pre>
bash rainbow.sh
</pre>
